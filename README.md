# THE-BLACKSHEEP
The BlackSheep Ink is a modern, fully responsive portfolio website built for Moeketsi Lehlohonolo, a professional tattoo artist specializing in fine-line realism, black & grey hyperrealism, and custom illustrative designs. The site serves as a digital showcase of his artistry, studio identity, and client engagement hub.

Key Features & Design Highlights:

Dynamic Theme Switcher – Visitors can toggle between dark and light modes, with the preference saved locally for a personalized browsing experience.

Fixed, Unclickable Header Navigation – A sleek glass-morphism header that stays out of the way visually while providing fully functional navigation links (Home, About, Service, Contacts) and a theme toggle button.

Hero Section – Bold animated typography with a typed.js effect (“Tattoo Artist / Custom Designer / Ink Storyteller”), a prominent call-to-action button, and social media icons linking to Facebook, WhatsApp, Instagram, and TikTok.

About Section – Side‑by‑side layout (image + text) that remains adjacent on desktop and gracefully stacks on mobile. Includes artist biography, experience tag, contact details, and a WhatsApp consultation button.

Services & Gallery – A two‑column layout: left sidebar lists signature tattoo styles (Fine Line, Realism, Cover‑Up, Lettering, Neo‑Traditional), while the right side displays a responsive image gallery. Each gallery card links to a dedicated detail page with a full‑size view and download button for the artwork.

Contact Section – Clean card displaying email, studio address, WhatsApp, and direct phone number, with a stylised quote reinforcing the brand’s identity.

Footer – Contains studio info, quick links, business hours, and social media icons with hover animations.

Performance & Interactivity – Smooth scrolling, scroll‑spy active link highlighting, Vanilla Tilt effect on images, ScrollReveal animations for section entries, and lazy loading for fast page loads.

Dedicated Work Detail Pages – Each gallery image opens an individual HTML page (work1.html … work10.html) with the artwork centered, a download button, and consistent navigation back to the main site – all matching the dark/light theme.

Technical Stack: HTML5, CSS3 (CSS variables for theming), JavaScript (jQuery, Typed.js, Vanilla Tilt, ScrollReveal), Font Awesome icons, and Google Fonts (Inter, Plus Jakarta Sans, Space Grotesk).

The website successfully communicates the artist’s brand – edgy, professional, and creative – while providing an intuitive, visually engaging experience for potential clients to explore his portfolio and easily initiate contact.
